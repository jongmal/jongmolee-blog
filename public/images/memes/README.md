# 개발자 밈 에셋

트러블슈팅 포스트에 삽입할 개발자 밈 이미지.

## 사용법

### 밈 추가
1. `untagged/`에 이미지 업로드 (파일명 = 밈 설명, 예: `세미콜론-하나-때문에.webp`)
2. 에이전트가 분류 → `tagged/`로 이동 + `index.json` 메타데이터 추가

### 태그 목록
| 태그 | 사용 시점 |
|------|----------|
| `bug-found` | 에러/버그 발견 순간 |
| `debugging` | 원인 추적, 삽질 중 |
| `root-cause` | 원인 발견, 허탈/충격 |
| `fixed` | 해결 성공, 쾌감 |
| `facepalm` | 과거 코드 보고 자괴감, 허탈 |
| `prevention` | 예방/교훈 |

### 트러블슈팅 글 매핑
```
## 🔍 증상     → bug-found
## 🧠 원인     → debugging / root-cause
## 🛠️ 해결     → fixed / facepalm
## 🔎 예방     → prevention
```

## 파일 형식
- webp 권장 (용량 최소화)
- 최대 500KB
- 가로 800px 이하 권장
