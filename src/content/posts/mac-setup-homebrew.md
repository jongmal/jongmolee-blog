**#1. 홈브루(Homebrew) 설치와 자동화의 시작**

안녕하세요! 이 글을 찾는 분들이시라면 맥북을 좀 더 편하게 쓰기 위해 방법을 찾고 계신 분들일 거라 생각합니다. 대학생, 전공자, 혹은 취미 개발자, 아니면 그저 맥북을 예쁘고 편리하게 꾸미는 '드레스 업'을 좋아하시는 분들 모두 환영합니다.

이번 포스팅에서는 기본적인 **CLI(Command Line Interface)**, 즉 터미널을 사용합니다. 검은 화면에 글자만 나오는 모습이 처음에는 낯설고 어려워 보일 수 있지만, 차례대로 따라오시다 보면 마우스로 일일이 클릭하는 것보다 이 방법이 훨씬 빠르고 익숙해지실 겁니다!

저는 개인적으로 맥에서 불편한 점이 아직 많아요. 윈도우, 리눅스, 맥 세 가지 환경을 돌아가면서 쓰다 보니 개발자 입장에서는 GUI(그래픽 인터페이스)가 낯설기도 하고요. 하지만 반대로 맥에만 있는 독보적인 패키지들도 많아서 다른 OS를 쓸 때 역으로 불편함을 느끼기도 합니다 (요즘엔 그 격차가 줄고 있긴 합니다만).

---

## 1. 홈브루(Homebrew)란 무엇인가요?

쉽게 말해 홈브루는 **"터미널판 앱스토어"**라고 생각하시면 됩니다.

![Homebrew 공식 사이트](/images/mac-setup-homebrew/20260306204421.png)

> [brew.sh](https://brew.sh/) — Homebrew 공식 사이트

원래 맥에서 프로그램을 깔려면 브라우저를 열고, 홈페이지를 찾아 들어가서, 설치 파일을 다운로드하고, 드래그 앤 드롭으로 설치하는 과정을 거쳐야 하죠. 하지만 홈브루가 있다면 터미널에 명령어 한 줄만 입력하는 것으로 이 모든 과정이 끝납니다. 수많은 앱을 하나씩 찾아다니며 깔 필요가 없게 만들어주는 마법 같은 도구죠.

## 2. 홈브루 설치 및 셋업

맥 세팅 자동화의 뼈대를 잡기 위해 홈브루를 먼저 설치해 보겠습니다.

### ① 홈브루 설치하기

먼저 맥의 **터미널(Terminal)** 앱을 실행한 후, 아래의 공식 설치 스크립트를 복사해서 붙여넣고 엔터를 누르세요.

![cmd + 스페이스바로 Spotlight 실행 후 터미널 검색](/images/mac-setup-homebrew/20260306205509.png)

![터미널에서 홈브루 설치 시 비밀번호 입력창](/images/mac-setup-homebrew/20260306205909.png)

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

> **tip.** 설치 중 비밀번호를 물어보면 맥 로그인 비밀번호를 입력하세요. 글자가 화면에 보이지 않아도 입력되고 있는 것이니 당황하지 마세요!

### ② 환경 변수(.zshrc) 등록하기

최신 맥은 기본적으로 `zsh` 쉘을 사용합니다. 설치가 끝나면 터미널이 "Next steps"라며 몇 가지 명령어를 더 입력하라고 안내할 텐데, 이를 통해 시스템이 `brew`라는 명령어를 알아먹을 수 있게 등록해줘야 합니다.

![path 설정 코드와 설정 적용 명령어 입력](/images/mac-setup-homebrew/20260306210453.png)

```bash
# 내 계정의 zsh 설정 파일에 홈브루 경로 추가
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc

# 변경된 설정 바로 적용
source ~/.zshrc
```

이제 터미널에 `brew -v`라고 입력했을 때 버전 정보가 나온다면 성공적으로 설치된 것입니다!

## 3. 간단 사용법 및 설치 테스트

홈브루가 잘 작동하는지 확인하기 위해, 맥 사용자들의 필수 유틸리티인 **Rectangle**(창 분할 관리 도구)을 설치해 보겠습니다.

> [rectangleapp.com](https://rectangleapp.com/) — 단축키로 빠르게 분할화면을 이동하는 가장 인기 있는 패키지입니다.

![단축키 설정부터 개인이 원하는 레이아웃 설정이 가능](/images/mac-setup-homebrew/20260306210828.png)

**명령어 입력:**

![설치 명령어와 설치 완료 과정](/images/mac-setup-homebrew/20260306211019.png)

```bash
brew install --cask rectangle
```

![응용 프로그램에 Rectangle 설치 완료](/images/mac-setup-homebrew/20260306211216.png)

**결과 확인:** 설치가 완료되면 응용 프로그램 폴더에 Rectangle이 생성된 것을 볼 수 있습니다. 마우스 클릭 한 번 없이 프로그램 설치가 끝났습니다! 이런 방식으로 앞으로 우리가 필요한 모든 앱을 한 번에 설치할 수 있게 됩니다.

![첫 실행 이후 상태바에 표시](/images/mac-setup-homebrew/20260306211317.png)

첫 실행 이후에는 상태바에 추가되어, 현재 셋업과 세팅 변경이 가능합니다.

**추가 팁 (자동 업데이트 설정):** 매번 업데이트를 체크하기 귀찮다면 `homebrew-autoupdate`를 활용해 보세요. 컴퓨터를 켤 때나 24시간마다 자동으로 업데이트와 최적화가 진행되도록 설정할 수 있습니다.
