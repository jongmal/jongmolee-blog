맥 초기 세팅 자동화 시리즈 두 번째 포스팅, **맥북 드레스업 패키지 소개** 편이에요. 이번엔 맥의 외관과 메뉴 바, 환경을 꾸미는 도구들에 집중해요.

---

**#2. 내 맥북을 더 특별하게, 드레스업(UI & 환경) 패키지 추천**

지난 편에서 홈브루(Homebrew) 설치를 마쳤다면, 이제 그 위에 예쁘고 실용적인 옷을 입혀줄 차례예요. 뼈대는 잡혔으니 이제 감성을 더할 시간이에요.

맥북을 선택하는 이유 중 하나가 깔끔한 UI이죠. 그런데 쓰다 보면 메뉴 바가 아이콘으로 꽉 차거나, 외부 모니터 설정이 번거로울 때가 있어요. **터미널 명령어 한 줄**로 맥북의 감성과 편의성을 동시에 잡을 수 있어요.

## 1. 🗂️ 깔끔한 메뉴 바의 완성 (노치 대응)

최신 맥북의 노치 때문에 메뉴 바 아이콘이 가려지는 건 진짜 불편한 문제예요. 아래 두 앱이 확실한 해결책이에요.

- **Bartender 5:** 메뉴 바 아이콘을 숨기거나 그룹화해 깔끔하게 관리해 줘요. 배터리 잔량이나 특정 앱 실행 시에만 아이콘이 표시되도록 세밀하게 설정할 수 있어요. 단축키 토글 기능도 있는데 (솔직히 이게 제일 편함), 한 번 쓰면 못 놓게 되더라고요. 유료지만 충분히 값을 해요.

[Bartender 5 공식 사이트](https://www.macbartender.com/Bartender5/)에서 30일 무료 체험도 가능해요.

![Bartender 스왑 기능](/images/mac-setup-dressup/20260306231402.png)

![Hidden Bar 앱스토어](/images/mac-setup-dressup/20260306231410.png)

**설치 명령어:** `brew install --cask bartender`

> **팁:** 30일 무료 체험이 가능해요. 먼저 써보고 결제를 결정해도 전혀 늦지 않아요.

- **Hidden Bar:** Bartender의 가벼운 무료 대안이에요. 토글 바를 기준으로 왼쪽 아이콘을 클릭 한 번에 숨겨주는 방식이에요. 세밀한 설정은 없지만, 간단히 정리만 원한다면 이걸로 충분해요.

[App Store에서 Hidden Bar 보기](https://apps.apple.com/kr/app/hidden-bar/id1452453066?mt=12)

![Hidden Bar 스크린샷](/images/mac-setup-dressup/20260306231747.png)

**설치 명령어:** `brew install --cask hiddenbar`

---

## 2. 🔤 폰트도 터미널로? (글꼴 설치 자동화)

디자인의 완성은 서체예요. 폰트 파일을 일일이 내려받아 서체 관리자에 등록할 필요 없이, 홈브루로 인기 서체를 바로 설치할 수 있어요. 🎉

- **Pretendard (프리텐다드):** 가독성이 뛰어나 맥 사용자들에게 가장 사랑받는 서체예요. 한국어와 영문 모두 균형 잡힌 자형이라 어디에 써도 자연스러워요.

**설치 명령어:** `brew install --cask font-pretendard`

![스팟라이트에서 서체 관리자 검색](/images/mac-setup-dressup/20260306232548.png)

![서체관리자에서 설치된 폰트 확인](/images/mac-setup-dressup/20260306232626.png)

*맥북 시스템 폰트는 순정 상태에서 변경이 불가능해요. 폰트를 지원하는 앱(문서 편집기, 에디터 등)에서만 적용 가능해요.*

> **주의:** 시스템 전체 폰트를 교체하려면 별도 수정 방법이 필요해요. 일반 사용 환경에서는 앱별로 적용하는 게 안전해요.

## 3. ⚡ 업무 효율을 높여주는 UI 유틸리티

작지만 강력한 도구들로 업무 환경을 쾌적하게 만들어 봐요.

- **Itsycal:** 메뉴 바에 아주 작은 달력을 띄워줘요. 메뉴 바 시계를 클릭하면 미니 달력이 팝업으로 나타나요. 캘린더 앱과 연동돼서 일정도 바로 확인할 수 있어요.

![Itsycal 스크린샷](/images/mac-setup-dressup/20260306232817.png)

- **MonitorControl:** 맥북 키보드의 밝기·음량 키로 외부 모니터까지 제어해요. DDC(디스플레이 데이터 채널) 프로토콜을 이용해 모니터 자체 밝기를 조절하는 방식이에요. 외부 모니터를 쓴다면 진짜 필수예요. ✨

![MonitorControl 스크린샷](/images/mac-setup-dressup/20260306232832.png)

- **HazeOver:** 작업 중인 창 외의 배경을 어둡게 만들어 집중력을 극대화해 줘요. 어둡기 정도를 슬라이더로 조절할 수 있어서 취향껏 설정 가능해요.

![HazeOver 스크린샷](/images/mac-setup-dressup/20260306233013.png)

**설치 명령어:** `brew install --cask itsycal monitorcontrol hazeover`

> **핵심:** MonitorControl은 외부 모니터 사용자에게 특히 강력 추천해요. 모니터 OSD 버튼 없이 키보드로 바로 밝기 조절이 돼요.

## 4. 🩺 내 맥북의 건강 상태 모니터링

맥북의 하드웨어 상태를 실시간으로 체크하는 도구들이에요.

- **Macs Fan Control:** 맥북의 온도를 확인하고 팬 속도를 수동으로 조절해요. 고사양 작업 중 발열이 심할 때 팬을 미리 높게 설정해두면 온도 관리가 훨씬 편해요.

- **coconutBattery:** 배터리의 효율, 충·방전 사이클, 제조일 등 상세한 건강 정보를 보여줘요. 중고 맥북 살 때 이걸로 배터리 상태 먼저 확인하는 게 국룰이에요 ㅋㅋ

**설치 명령어:** `brew install --cask macs-fan-control coconutbattery`

> **팁:** coconutBattery에서 배터리 최대 충전 용량을 꼭 확인해 보세요. 설계 용량 대비 **80% 이하**면 교체를 고려할 시점이에요.
